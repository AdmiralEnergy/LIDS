# Compass

Mobile PWA for field sales representatives.

## Features

- **Agent Chat**: Conversational AI assistant
- **Lead Enrichment**: Property lookup
- **Offline Mode**: Works without internet
- **PWA**: Installable on mobile

## Known Issue

Agent responses are currently mocked. To connect to real COMPASS agents, update `server/routes.ts` to proxy to `192.168.1.23:4098`.
